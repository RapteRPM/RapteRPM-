        <p>- Problematica: <br> En los últimos años se ha venido viendo un gran incremento a las estafas digitales del 134% en Colombia. Esto debido a que las plataformas usadas comúnmente para la compra y ventas, no poseen un riguroso control de los usuarios, permitiendo así la creación de cuentas falsas. Este Proyecto pretende implementar tecnología más segura para el debido control de usuarios estableciendo así una infraestructura de seguridad más robusta.</p>
        <p>- Objetivo del Proyecto: <br>El objetivo del proyecto RaptoRPM es generar una interacción segura, eficiente y confiable entre compradores y vendedores que facilite la comercialización y adquisición de moto partes en la ciudad de Bogotá y así promover el crecimiento sostenible del negocio. A través del desarrollo de un sistema de información web con un meticuloso proceso de creación de perfiles verificados, además de realizar una herramienta de búsqueda y comunicación que complemente la interacción entre comprador y vendedor tanto en la búsqueda de autopartes como en adquirir servicios mecánicos.</p>
        <p>- Objetivos específicos: <br>1. Optimizar una interfaz para que el usuario navegue en la página web donde pueda buscar, comparar y comprar moto partes de manera rápida y fácil.	
            <br>2. Implementar una forma de intercambio seguro de moto partes entre vendedores y compradores mediante un proceso de creación de cuentas verificadas para ambos tipos de usuarios.
            <br>3. Implementar medidas de seguridad en pro de proteger la integridad e información del cliente donde se garantice la seguridad en la información. 
            <br>4. Implementar un sistema de reseñas y/o calificaciones en vendedores para que así se fomente la confianza entre vendedores y compradores.
            <br>5. Promocionar los negocios locales que se destaquen por su buen desempeño en ventas a través de la aplicación.
